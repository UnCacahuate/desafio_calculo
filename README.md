# 🛒 Desafío: Cálculo de Total

Ejercicio introductorio de **JavaScript** para practicar la manipulación del DOM y el manejo de eventos.

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript

## 🎯 Funcionalidades

- Mostrar el precio de un producto.
- Aumentar y disminuir la cantidad.
- Calcular el total a pagar en tiempo real.

## 📚 Conceptos practicados

- `querySelector()`
- `innerHTML`
- Eventos `onclick`
- Funciones
- Conversión de datos con `Number()`
